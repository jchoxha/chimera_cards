**Card Editor adds an Editor button to the main menu that opens a Compendium-style card browser, letting you edit cards directly in-game.**

You can edit core card properties such as energy cost, star cost, replay count, and card type. The mod also lets you add or remove keywords like Exhaust, Ethereal, Retain, and others, as well as change dynamic values such as damage, block, and similar card numbers.

It supports Enchantments and Afflictions with adjustable amounts, and also allows you to add reusable Extra Effects with configurable trigger, target, timing, and duration. Each Extra Effect can also have its own scaling, based on card history such as plays, draws, discards, exhausts, or cards created.

The mod includes a preset system with save, load, delete, and Revert to Vanilla options. There is also a hotkey for quickly opening the Editor, set to F11 by default, and this can be rebound in the menu. In addition, the Creator feature allows custom card slots and optional custom art loaded from disk.

It also includes a low profile Relic Editor that is mostly Experimental for now

Installation:

Installation works on both the old loader and the beta/new mod loader. Create the folder ...\\Slay the Spire 2\\mods\\Card\_editor\ and place card\_editor.dll and card\_editor.pck inside it. You should also include card\_editor.json, which is required for the beta/new mod loader. After that, launch the game and open the Editor from the main menu.

Instructions:

If you want to use custom portrait art with the Creator, create a folder next to card\_editor.dll called either card art or card\_art. Drop image files into that folder using png, jpg, jpeg, or webp format. The recommended portrait size is 1000x760 or higher, matching the base game portrait size. In the Creator’s Art dropdown, these will appear as Custom: filename.

Presets are stored in user://card\_editor/presets/\*.json. Logs can be found in user://logs/godot.log, and you can search for \[CardEditor\] there. Presets and other JSON files are also stored in the Slay the Spire 2 roaming folder. The total number of custom cards can be edited through created\_cards.json in that roaming folder.

The mod includes a preset system with save, load, delete, and Revert to Vanilla options. There is also a hotkey for quickly opening the Editor, set to F11 by default, and this can be rebound in the menu. In addition, the Creator feature allows custom card slots and optional custom art loaded from disk.

You can also press F11 to open the Editor during combat, or rebind the key in the menu if needed.
